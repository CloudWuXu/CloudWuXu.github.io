自我介绍

我叫Cloud云，来自广东顺德CC镇，毕业于南方普通大学。
我兴趣广泛，平常喜欢运动、阅读、动漫、游戏和编程。
目前，从事光学成像系统设计，本职光学工程师，偶尔客串结构和工艺。

写博客的初衷

记录自己学习、工作以及生活的痕迹。

兴趣

运动 : 足球 篮球 羽毛球

生命不止 运动不息

阅读 : 推理 悬疑

推理小说是我的挚爱，入坑作《白夜行》，从此一发不可收拾。其他类型的相对较少。
柯南道尔、阿加莎克里斯蒂以及东野圭吾这三位作家的作品看得比较多，宫部美雪和紫金陈的略有涉猎。
推荐书目
1）柯南道尔：《血字的研究》《四签名》《巴斯克维尔的猎犬》
2）阿加莎克里斯蒂：《东方快车谋杀案》《尼罗河上的惨案》《罗杰疑案》
3）东野圭吾：《白夜行》《嫌疑人X的献身》《祈祷落幕时》
4）宫部美雪：《模仿犯》《火车》《理由》
5）紫金陈：《长夜难明》《坏小孩》《无证之罪》

动漫 : 热血 冒险

《Fairy Tail》《Bleach》《JOJO》《当哒当》排名不分先后

游戏 : 战略 RPG

《红警2》最早接触的RTS游戏
《红警3》虽然可选阵营只有三个，但不影响我对它的喜爱
《星际争霸2》在我心中是最好的RTS游戏，可是手残有点玩不来
《文明6》内容十分丰富，玩法多样，一局游戏，恍如隔世
《口袋妖怪系列》GBA模拟上的经典游戏
《仙剑奇侠传4》云通关了，剧情系列之最
《最终幻想7重制版》希望21年能玩上PC版
《艾尔登法环》
《黑神话：悟空》

编程 : Github Python Matlab C++

本着能抄坚决不改、能改坚决不编的原则，我会从Github或B站上找感兴趣的项目去实现。
一路上走来，踩了不少坑，唯一的心得就是多查资料，多实践。
有时难免需要自己写代码，简单的用Python处理，复杂的先Matlab后C++。
